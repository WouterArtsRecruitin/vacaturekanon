## 📞 ESCALATION PROTOCOL

**If CPL exceeds €12:**
1. Pause campaign immediately (same day)
2. Document: What changed? (targeting, audience, seasonality?)
3. Review creative: Is messaging still relevant?
4. Review landing page: Is form working? Load time OK?
5. Re-test with new creative variants
6. Restart campaign when ready

**If No Leads for 24 Hours:**
1. Check campaign status: Still "ACTIVE"?
2. Check budget: Is it running out?
3. Check targeting: Audience size still > 10K?
4. Check platform: Any Meta API issues?
5. Contact Meta Support if needed
