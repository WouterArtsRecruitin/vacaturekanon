## 📞 SUPPORT & TROUBLESHOOTING

**Ads not approved?**
- Check for text overlay (max 20%)
- Verify no misleading claims
- Remove urgency language if flagged

**Low CTR after 3 days?**
- Check audience size (need min 1k people)
- Try different images (rotate to variant B/C)
- Increase bid by 10%

**Pixel not firing?**
- Verify pixel ID in Meta Ads Manager
- Check website has pixel code installed
- Test with Meta Pixel Helper (Chrome extension)

**Budget depleting too fast?**
- Check bid strategy (lower CPC target)
- Narrow audience (more specific targeting)
- Reduce bid by 20-30%
