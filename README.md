# Vacaturekanon 🎯
Recruitin B.V. — Automated recruitment landing page generator

## Flow
Jotform → Render Webhook → Claude Sonnet → Netlify Deploy → Pipedrive Deal

## Stack
- Webhook: Python/Flask op Render
- HTML Generator: Claude Sonnet 4.6
- Hosting landing pages: Netlify (per vacature)
- CRM: Pipedrive
- Notifications: Slack
